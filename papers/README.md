# Paper Bundle for TC6544 Project 2

This folder collects open-access papers for the literature review on Genetic
Algorithms for the Traveling Salesman Problem, permutation operators, repair,
and constraint handling.

Downloaded PDFs were taken from open arXiv PDF endpoints. Older classic papers
that are useful for the report but not clearly available as open PDFs are listed
under "Manual or library access".

## Downloaded PDFs

| File | Paper | Source | Project use |
|---|---|---|---|
| `abdoun-2012-adaptive-crossover-operators-tsp.pdf` | Abdoun and Abouchabaka, "A Comparative Study of Adaptive Crossover Operators for Genetic Algorithms to Resolve the Traveling Salesman Problem" | https://arxiv.org/abs/1203.3097 | Empirical comparison of TSP crossover operators. |
| `abdoun-2012-mutation-operators-tsp.pdf` | Abdoun, Abouchabaka, and Tajani, "Analyzing the Performance of Mutation Operators to Solve the Travelling Salesman Problem" | https://arxiv.org/abs/1203.3099 | Supports mutation-operator discussion. |
| `scholz-2019-ga-tsp-historical-review.pdf` | Scholz, "Genetic Algorithms and the Traveling Salesman Problem a historical Review" | https://arxiv.org/abs/1901.05737 | Background and historical framing for GA-TSP work. |
| `uray-2023-csrx-crossover-tsp.pdf` | Uray, Wintersteller, and Huber, "CSRX: A novel Crossover Operator for a Genetic Algorithm applied to the Traveling Salesperson Problem" | https://arxiv.org/abs/2303.12447 | Modern crossover-operator reference. |
| `thanh-2020-combination-crossover-ga-tsp.pdf` | Thanh, Binh, and Lam, "New mechanism of combination crossover operators in genetic algorithm for solving the traveling salesman problem" | https://arxiv.org/abs/2001.11590 | Additional crossover comparison and TSPLIB experimentation. |
| `cicirello-2023-survey-evolutionary-operators-permutations.pdf` | Cicirello, "A Survey and Analysis of Evolutionary Operators for Permutations" | https://arxiv.org/abs/2311.14595 | Broad survey of permutation crossover and mutation operators. |
| `liu-2014-powerful-genetic-algorithm-tsp.pdf` | Liu, "A Powerful Genetic Algorithm for Traveling Salesman Problem" | https://arxiv.org/abs/1402.4699 | GA-TSP benchmark reference with local-search/recombination context. |
| `chehouri-2016-constraint-handling-violation-factor-ga.pdf` | Chehouri, Younes, Perron, and Ilinca, "A Constraint-Handling Technique for Genetic Algorithms using a Violation Factor" | https://arxiv.org/abs/1610.00976 | Constraint-handling comparison against penalty-style methods. |
| `padhye-2015-feasibility-preserving-constraint-handling.pdf` | Padhye, Mittal, and Deb, "Feasibility Preserving Constraint-Handling Strategies for Real Parameter Evolutionary Optimization" | https://arxiv.org/abs/1504.04421 | Feasibility-preserving and repair-style constraint handling. |
| `eremeev-2017-optimal-recombination-atsp.pdf` | Eremeev and Kovalenko, "Genetic Algorithm with Optimal Recombination for the Asymmetric Travelling Salesman Problem" | https://arxiv.org/abs/1706.06920 | Recombination-based GA reference for TSP variants. |
