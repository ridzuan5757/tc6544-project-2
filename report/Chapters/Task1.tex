\noindent
The selection of kroA100 is justified based on four criteria. First, this dataset satisfies the required problem size constraint of $70 \leq n \leq 150$, providing sufficient complexity to demonstrate the behaviour of the

Next, this dataset belongs to the \textbf{random Euclidean class}, where cities are randomly distributed across a two-dimensional coordinate space that producing a well-spread layout that presents a balanced challenge for the Genetic Algorithm.
